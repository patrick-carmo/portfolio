Marriage

Um PWA que permitiu aos convidados de um casamento gravar mensagens em vídeo, fotos ou textos, para que o casal pudesse visualizá-las após a cerimônia.

O objetivo do projeto foi dar a possibilidade dos convidados poderem gravar vídeos, fotos ou enviar mensagens aos noivos.

Principais funcionalidades:

  • Autenticação com Google E JWT.
  • Persistência de daods com PostgreSQL.
  • Upload de arquivos com multer e Google Drive API.
  • Painel administrativo para controle total sobre mensagens, fotos e vídeos recebidos.
  • WebScoket para atualização em tempo real da barra de progresso.
  • Scroll infinito e Refresh para uma visualização mais fluida das mensagens.
  • Progressive Web App - Funciona tanto na web quanto no aplicativo mobile, consultando a mesma API.

Principais Tecnologias

  • TypeScript
  • Node.js
  • Nest.js
  • PostgreSQL
  • TypeORM
  • JWT
  • WebSocket
  • Ionic
  • Angular
  • Google Drive API

Sistema PDV

API Rest para um sistema de ponto de vendas.

O objetivo do projeto foi desenvolver uma API Rest para um sistema de ponto de vendas, fornecendo recursos essenciais para operação de vendas e gerenciamento de clientes, produtos e pedidos.

O desafio principal foi a organização do projeto, já que foi desenvolvido por mim e mais outros 3 desenvolvedores. O projeto foi entregue seguindo a metodologia ágil Scrum, com sprints de 3 semanas.

Principais funcionalidades:

  • Autenticação por JWT.
  • Senhas criptografadas com bcrypt.
  • Uso do Joi para valização dos dados de entrada.
  • Endpoints para CRUD relacionados a usuários, clientes, produtos e pedidos.
  • Upload das fotos dos produtos na núvem utilizando multer.
  • Utilização do nodemailer para enviar e-mails de confirmação para os clientes após a conclusão do pedido.

Principais Tecnologias

  • TypeScript
  • Node.js
  • Express.js
  • PostgreSQL
  • Knex
  • JWT
  • S3 AWS

IBGE News

PWA para visualização de notícias do IBGE.

O projeto foi realizado com outros 4 desenvolvedores. O objetivo do projeto foi criar uma aplicação com funcionalidades como autenticação via Firebase, consumo de uma API e exibição paginada dos dados obtidos, além da criação de apk funcional para android. Optamos por utilizar a API do IBGE Notícias.

Principais funcionalidades:

  • Autenticação com Google ou e-mail e senha.
  • Autenticação biométrica no app mobile.
  • Feed com as notícias apresentadas em cards, além de um scroll infinito e refresh.
  • Persistência de dados no banco de dados NoSQL Firestore.
  • Bookmarks, possibilitando o usuário salvar as notícias favoritas.
  • Sistema comentários com atualização em tempo real.
  • Sistema de likes.
  • Sistema de buscas.
  • Progressive Web App - Funciona tanto na web quanto no aplicativo mobile, consultando a mesma API.

Principais Tecnologias

  • TypeScript
  • Ionic
  • Angular
  • Firebase
  • Firestore
